# Analitika učenja

Projekt ima:

- Flask backend u rootu projekta
- Vue frontend u `frontend-app/`

## Pokretanje backenda

```bash
python3 -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
python main.py
```

Backend radi na:

```text
http://127.0.0.1:5005
```

## Pokretanje frontenda

```bash
npm install
npm run dev
```

Frontend radi na:

```text
http://127.0.0.1:3000
```

## Redoslijed

1. Pokreni backend
2. Pokreni frontend
3. Otvori `http://127.0.0.1:3000`
